inDepth deploy folder

This folder is ready for static hosting.

Recommended quickest path:
1. Go to Cloudflare Pages direct upload:
   https://developers.cloudflare.com/pages/get-started/direct-upload/
2. Choose drag-and-drop upload.
3. Upload this entire `deploy` folder.
4. Your public site will be available on a `pages.dev` URL you can open on your phone.

Files included:
- index.html
- demo.css
- demo-app.js
- indepth-logo.PNG

This deploy build opens directly to the inDepth app at the site root.
